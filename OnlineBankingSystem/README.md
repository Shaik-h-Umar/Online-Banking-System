# Online Banking System

This is a simple online banking system built with PHP and MySQL.

## Features

* User registration and login
* Account dashboard
* Money transfer
* Transaction history
* Profile management
* Deposit money

## Requirements

* PHP
* MySQL
* Apache or any other web server

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/OnlineBankingSystem.git
   ```
2. Create a database named `securebank`.
3. Import the `database.sql` file into your `securebank` database:
   ```bash
   mysql -u your-username -p securebank < database.sql
   ```
4. Move the project to your web server's root directory (e.g., `htdocs` for XAMPP).
5. Open the application in your browser.

## Usage

1. Register a new account.
2. Log in with your credentials.
3. You will be redirected to the dashboard.
4. You can now use the various features of the application.
